O que é JavaScript?

JavaScript é uma linguagem de programação usada para criar interatividade em páginas web. Com ele, você pode manipular o conteúdo HTML, estilizar dinamicamente elementos CSS e adicionar funcionalidades como respostas a eventos do usuário.

Estrutura Básica de um Código JavaScript

O JavaScript pode ser incluído diretamente dentro de uma página HTML usando a tag <script>.


<script>
  // Este é um comentário em JavaScript
  console.log("Olá, mundo!");  // Exibe "Olá, mundo!" no console do navegador
</script>

Variáveis

Em JavaScript, você pode armazenar valores usando variáveis. Existem três maneiras principais de declarar uma variável: var, let e const.


let nome = "João";  // Variável mutável
const idade = 30;   // Constante (não pode ser alterada)
var cidade = "São Paulo";  // Var (antigo, menos usado)

Funções

Funções em JavaScript permitem agrupar código reutilizável em blocos que podem ser chamados quantas vezes forem necessários.


function saudacao(nome) {
  return "Olá, " + nome + "!";
}

console.log(saudacao("Maria"));  // Exibe "Olá, Maria!"

Eventos

JavaScript pode responder a eventos, como cliques de botões ou o carregamento de uma página.


document.getElementById("meuBotao").addEventListener("click", function() {
  alert("Botão clicado!");
});

Manipulação do DOM

O DOM (Document Object Model) representa a estrutura de uma página web como um conjunto de objetos manipuláveis por JavaScript. Com o DOM, você pode acessar e modificar elementos HTML dinamicamente. 


document.getElementById("titulo").innerHTML = "Novo Título";

Condições e Laços

Condições

Instruções condicionais permitem que você controle o fluxo do programa.


let numero = 10;

if (numero > 5) {
  console.log("O número é maior que 5");
} else {
  console.log("O número é menor ou igual a 5");
}

Laços

Os laços permitem a repetição de uma parte do código várias vezes.


for (let i = 0; i < 5; i++) {
  console.log("Número " + i);
}

Arrays e Objetos

JavaScript permite armazenar múltiplos valores em uma única variável usando arrays ou objetos.

Array


let frutas = ["Maçã", "Banana", "Laranja"];
console.log(frutas[0]);  // Exibe "Maçã"

Objeto


  let pessoa = {
    nome: "Ana",
    idade: 25,
    cidade: "Rio de Janeiro"
  };
  console.log(pessoa.nome);  // Exibe "Ana"

Promessas e Assíncrono

JavaScript é conhecido por suas capacidades assíncronas. Promises são usadas para lidar com operações que podem demorar, como requisições a servidores.


  fetch('https://api.exemplo.com/dados')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.log('Erro:', error));

Mais Recursos JavaScript

JavaScript é uma linguagem extremamente poderosa, com inúmeras funcionalidades para você explorar. Continue aprendendo com exemplos práticos e experimentando no seu navegador!

Exemplos de Uso

Aqui estão alguns exemplos de como você pode usar JavaScript em sua página web:

Exemplo 1: Contador de Cliques


  let contador = 0;
  
  document.getElementById("botao").addEventListener("click", function() {
    contador++;
    document.getElementById("contador").innerHTML = "Cliques: " + contador;
  });

Exemplo 2: Formulário de Contato


  document.getElementById("formulario").addEventListener("submit", function(event) {
    event.preventDefault();
    let nome = document.getElementById("nome").value;
    let email = document.getElementById("email").value;
    let mensagem = document.getElementById("mensagem").value;
    console.log("Nome: " + nome);
    console.log("Email: " + email);
    console.log("Mensagem: " + mensagem);
  });

Exemplo 3: Galeria de Imagens


  let imagens = ["imagem1.jpg", "imagem2.jpg", "imagem3.jpg"];
  
  document.getElementById("galeria").addEventListener("click", function() {
    let imagemAtual = document.getElementById("imagem-atual").src;
    let indice = imagens.indexOf(imagemAtual);
    if (indice >= 0) {
      indice++;
      if (indice >= imagens.length) {
        indice = 0;
      }
      document.getElementById("imagem-atual").src = imagens[indice];
    }
  });
