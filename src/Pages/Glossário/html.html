O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas web. Ela permite estruturar o conteúdo de uma página, criando cabeçalhos, parágrafos, links, imagens e muito mais. Abaixo, você encontrará um glossário com os principais elementos de HTML, acompanhados de explicações e exemplos práticos.

Estrutura Básica de um Documento HTML

Todo documento HTML segue uma estrutura básica que é reconhecida por navegadores. Veja abaixo a estrutura mínima que uma página HTML deve ter: 


<!DOCTYPE html>  
<html>             
  <head>         
    <title>Título da Página</title>  
  </head>

  <body>         
    <h1>Cabeçalho Principal</h1>
    <p>Este é um parágrafo de exemplo.</p>
  </body>
</html>

Explicação dos Elementos:

  • <!DOCTYPE html>: Informa ao navegador que o documento segue o padrão HTML5.
  • <html>: A raiz do documento HTML. Todo o conteúdo da página deve estar entre as tags <html> e </html>.
  • <head>: Contém meta-informações sobre o documento, como o título, links para CSS e scripts, charset, etc.
  • <title>: Define o título da página, que aparece na aba do navegador.
  • <body>: É onde o conteúdo visível da página é colocado, como textos, imagens, links, etc.

Tags HTML Mais Comuns

<h1> a <h6> - Cabeçalhos

Os elementos de cabeçalho são usados para definir títulos e subtítulos em uma página. Eles variam de <h1> (mais importante) até <h6> (menos importante).


<h1>Cabeçalho Principal</h1>   
<h2>Subtítulo de Seção</h2>   
<h3>Subseção</h3>

<p> - Parágrafo

Define blocos de texto em um parágrafo. O navegador aplica automaticamente uma margem vertical antes e depois de um parágrafo.


<p>Este é um parágrafo de exemplo.</p>

<a> - Link

O elemento <a> (âncora) cria um link clicável.


<a href="https://www.exemplo.com">Clique aqui para visitar o Exemplo</a>
  • href: Define o destino do link, seja uma URL externa ou uma seção dentro da própria página.

<img> - Imagem

Incorpora imagens na página. Utiliza o atributo src para definir o caminho da imagem e alt para fornecer um texto alternativo.


<img src="imagem.jpg" alt="Descrição da imagem">
  • src: Define o caminho para a imagem que será exibida.
  • alt: Fornece um texto alternativo para a imagem, útil para acessibilidade e SEO.

Formatação de Texto

O HTML também oferece tags para formatar o texto dentro de uma página:

  • <b> ou <strong>: Deixa o texto em negrito (forte).
  • <i> ou <em>: Deixa o texto em itálico (ênfase).
  • <u>: Sublinha o texto.
  • <br>: Insere uma quebra de linha.

<p>Este é um texto <b>negrito</b>, <i>itálico</i> e <u>sublinhado</u>.</p>

Listas

<ul> - Lista Não Ordenada

Cria uma lista de itens com marcadores.


<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol> - Lista Ordenada

Cria uma lista de itens numerados.


<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Formulários

Formulários permitem a interação do usuário com a página, como envio de dados. Veja um exemplo de formulário simples:


<form action="/enviar" method="POST">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Enviar">
</form>

Mais Tags e Atributos

Esta é apenas uma introdução aos elementos principais do HTML. Existem muitas outras tags e atributos que permitem construir páginas web complexas e interativas.

Alguns exemplos de tags e atributos adicionais incluem:

  • <table>: Cria tabelas para organizar dados.
  • <video> e <audio>: Incorporam mídia de vídeo e áudio.
  • <canvas>: Permite a criação de gráficos e desenhos dinâmicos.
  • <svg>: Define gráficos vetoriais escaláveis.
  • <nav>, <header>, <footer>: Estruturam a página em seções lógicas.
  • <section>, <article>, <aside>: Organizam o conteúdo em seções temáticas.

Práticas Recomendadas

Para escrever HTML de qualidade, é importante seguir algumas práticas recomendadas:

  • Use tags semânticas: Em vez de usar apenas <div> e <span>, use tags que descrevam o conteúdo, como <header>, <nav>, <main>, etc.
  • Evite o uso excessivo de IDs: IDs devem ser únicos em uma página. Prefira usar classes para estilos e seletores.
  • Use atributos de acessibilidade: Atributos como alt, title e aria-* melhoram a acessibilidade do seu conteúdo.
  • Minimize o uso de tabelas para layout: Tabelas devem ser usadas para dados tabulares, não para layout de página.
  • Teste em diferentes navegadores e dispositivos: Certifique-se de que sua página funciona bem em diferentes ambientes.

Conclusão

Este guia ofereceu uma visão geral dos elementos básicos do HTML e como eles podem ser usados para criar páginas web. Lembre-se de que a prática leva à perfeição, então continue a aprender e a experimentar com diferentes tags e atributos para se tornar um desenvolvedor web habilidoso.
