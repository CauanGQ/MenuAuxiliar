O que é CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML. Ela define como os elementos HTML serão exibidos na tela, em papel ou em outros meios. Abaixo você encontrará um glossário com as principais propriedades e conceitos do CSS, juntamente com exemplos práticos.

Estrutura Básica de um Arquivo CSS

Um arquivo CSS define regras de estilo para elementos HTML. Cada regra tem um seletor (para indicar o que será estilizado) e um bloco de declarações (com as propriedades e seus valores). 


/* Este é um comentário em CSS */
selector {
  property: value;  /* Propriedade e valor */
}

Exemplo de Uso em uma Página HTML

Um arquivo CSS pode ser vinculado a uma página HTML usando a tag <link> no cabeçalho do documento.


<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <link rel="stylesheet" href="styles.css">  
  </head>
  <body>
    <h1>Título Estilizado</h1>
    <p>Este é um parágrafo com estilos aplicados.</p>
  </body>
</html>

Seletor de Elementos

O seletor mais básico no CSS é o seletor de elementos. Ele aplica um estilo a todas as ocorrências de um determinado elemento HTML.


h1 {
  color: blue;  /* Altera a cor do texto do h1 para azul */
}
p {
  font-size: 16px;  /* Define o tamanho da fonte dos parágrafos */
}

Classes e IDs

Você pode criar seletores mais específicos utilizando classes (com o símbolo ".") ou IDs (com o símbolo "#").


/* Estilo para uma classe */
.introducao {
  font-weight: bold;  /* Deixa o texto em negrito */
}

/* Estilo para um ID */
#titulo-principal {
  text-align: center;  /* Centraliza o texto */
}

No HTML, as classes e IDs são aplicadas assim:


<h1 id="titulo-principal">Título Centralizado</h1>
<p class="introducao">Texto em negrito.</p>

Propriedades de Layout

As propriedades de layout definem como os elementos são dispostos na página.


.container {
  display: flex;
  justify-content: space-between;  /* Distribui o espaço uniformemente */
}

.item {
  flex-basis: 200px;  /* Define a largura inicial do item */
}

Propriedades de Estilo

As propriedades de estilo definem a aparência dos elementos.


h1 {
  color: red;  /* Define a cor do texto como vermelho */
}

p {
  background-color: lightblue;  /* Define a cor de fundo como azul claro */
  padding: 20px;  /* Adiciona um espaçamento interno de 20px */
  border: 1px solid black;  /* Adiciona uma borda preta de 1px */
  border-radius: 10px;  /* Adiciona uma curvatura de 10px nas bordas */
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.5);  /* Adiciona uma sombra preta com 50% de opacidade */
}

Propriedades de Transição e Animação

As propriedades de transição e animação definem como os elementos mudam de estado.


h1 {
  transition: color 0.5s ease-in-out;  /* Define uma transição de cor com duração de 0.5s e efeito de entrada e saída suave */
}

p {
  animation: fade-in 2s infinite;  /* Define uma animação de fade-in com duração de 2s e repetição infinita */
}

@keyframes fade-in {
  0% {
    opacity: 0;  /* Define a opacidade inicial como 0 */
  }
  100% {
    opacity: 1;  /* Define a opacidade final como 1 */
  }
}

Propriedades de Responsividade

As propriedades de responsividade definem como os elementos se adaptam a diferentes tamanhos de tela.


@media (max-width: 600px) {
  h1 {
    font-size: 24px;  /* Define o tamanho da fonte como 24px em telas menores que 600px */
  }
}

@media (max-width: 400px) {
  p {
    font-size: 18px;  /* Define o tamanho da fonte como 18px em telas menores que 400px */
  }
}
